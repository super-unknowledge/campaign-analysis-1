# campaign-analysis-1
Social Media Campaign Performance analysis using a synthetic dataset from Real World Fake Data. This is part 1 of a 3 part series - Data Analysis. The dataset contains synthetic social media campaign data sourced from Real World Fake Data here: https://sonsofhierarchies.com/2025/08/01/rwfd-season-4-dataset-1-social-media-campaign-performance/ You can also find that I've uploaded a copy of the original dataset here in this repo.

### Process
The notebook goes over an end-to-end analysis of the dataset, covering cleaning, data exploration, A/B testing, and visualization.

### Analysis

**Which marketing campaigns are actually worth the money we spend on them?**

- Instagram gives the best return per dollar, with both high Cost per Click and high Return on Ad Spend. TikTok performs the worst, with a high CPC and low ROAS. The TikTok campaigns need to be reassessed.
- All 5 content themes (Product Launch, Behind-The-Scenes, User Testimonial, Lifestyle, Promotion) have an average conversion rate of about 12%. There are no standouts between the themes.
- Carousels on Instagram give the best average ROAS. Each platform has it's own best content type.
